# BlackOut
웹에서 플레이 가능한 어몽어스 스타일의 2D 멀티플레이 마피아 게임.


🎮 **[웹에서 바로 플레이하기](https://BlackOut-PJT.github.io/blckout/BlackOut)**

## Tech Stack
- **Engine**: Unity 3.16.3
- **Language**: C#
- **Network**: Photon PUN2
- **Platform**: WebGL (Github Pages 배포)

## 🕹️ 게임 방법
* **이동**: `W`, `A`, `S`, `D` 또는 `방향키`
* **상호작용**: `E` (아이템 줍기, 조합대 사용, 투표 시작)
* **생존자 승리조건**: 생존자들은 아이템을 조합해 폭죽을 3번 터뜨리거나 살인마를 투표를 통해 색출하여 승리
* **살인마 승리조건**: 투표나 살인을 통해 생존자를 모두 제거
